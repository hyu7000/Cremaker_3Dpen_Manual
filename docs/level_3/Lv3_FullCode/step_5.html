모터 작동

이제 실제로 모터를 작동하는 코드를 추가해보려 합니다. 모터는 온도가 예열된 상태에서만 작동해야합니다. 버튼을 눌러도 온도가 높지 않다면, 출력이 되지 않게끔 해야 합니다.
따라서 여기에서는 온도에 따라 모터가 작동되도록 코드를 구현하는 것이 목표입니다.

모터 작동 코드를 추가하기전에 현재온도가 목표온도에 근접했는지 확인할 수 있는 코드를 작성해주어야 합니다. 보통 상태를 확인하는 코드는 bool 변수를 사용하면 좋습니다.

※ 전체코드를 계속해서 보여주는 것은 코드가 길기 때문에 작성된 코드에서 추가될 부분만 설명드리겠습니다.

 1// 모터로 재료를 밀어낼 준비가 된 상태인지 확인하는 bool 변수 선언
 2bool isReadyToExt = false;
 3
 4// 현재 상태가 모터를 작동시킬수 있는지 확인하는 함수
 5void checkForReadyToExt()
 6{
 7    if(curTemp > setTemperature - 5)
 8    {
 9        isReadyToExt = true;
10    }
11    else
12    {
13        isReadyToExt = false;
14    }
15}
현재온도가 목표온도에서 5도 차이내에 있다면, 모터를 작동시킬 수 있도록 isReadyToExt 변수를 true로 변경해줍니다.
현재온도가 목표온도보다 높은 경우에는 모터가 작동하는데 문제가 없기 때문에, 온도가 낮은 경우만 고려하였습니다.

하지만 고려해야할 것이 하나 더 있습니다. 어떤 것이 문제가 있을지 한번 생각해보시고 아래 버튼을 눌러 수정된 코드를 확인해보세요.

 1// 모터로 재료를 밀어낼 준비가 된 상태인지 확인하는 bool 변수 선언
 2bool isReadyToExt = false;
 3
 4// 현재 상태가 모터를 작동시킬수 있는지 확인하는 함수
 5void checkForReadyToExt()
 6{
 7    // 설정온도가 0 이라면, isReadyToExt 변수를 false로 저장
 8    if(setTemperature == 0)
 9    {
10        isReadyToExt = false;
11        return;
12    }
13
14    if(curTemp > setTemperature - 5)
15    {
16        isReadyToExt = true;
17    }
18    else
19    {
20        isReadyToExt = false;
21    }
22}
문제는 설정온도가 0 이면, 현재온도보다 항상 낮은 상태가 되니 isReadyToExt 변수가 true 로 저장될 것입니다.
그렇다면 예열이 되지 않은 상태에서도 모터가 움직이게 됩니다.
위 함수에서 저장되는 isReadyToExt 변수를 사용하여 모터 작동코드를 작성하면 아래와 같습니다.
 1// 방향을 알기 위한 true, false를 전처리문으로 선언
 2#define EXT_DIR true
 3#define REV_DIR false
 4
 5/*
 6 * 방향과 속도를 정하여 모터를 작동시키는 함수, 0이면 정지, 255면 최대속도로 통일
 7 */
 8void workMotor(int speed, bool dir)
 9{
10    // 모터를 작동시킬 준비가 되지 않았다면, 함수를 종료
11    if(!isReadyToExt)
12    {
13        return;
14    }
15
16    if(dir==EXT_DIR)
17    {
18        digitalWrite(6, HIGH);
19        analogWrite(10, 255-speed); // HIGH일 경우 스피드를 반전
20    }
21    else if(dir==REV_DIR)
22    {
23        digitalWrite(6, LOW);
24        analogWrite(10,speed);
25    }
26}
모터를 작동하는 함수 앞에 isReadyToExt 의 변수를 확인하는 코드를 추가하여, 온도상태에 따라 모터를 작동시키거나 함수를 종료합니다.

3D 펜에서 재료를 밀어내거나 제거하는 모터의 동작은 다음과 같이 진행됩니다.


재료를 밀어내는데에는 밀어내다가 버튼을 손에서 떼면, 살짝 뒤로 당겨줍니다. 이는 필라멘트 재료의 특성 때문에 흘러내림을 방지하기 위함입니다. 이 동작을 리트렉션 이라고 합니다.

아래는 재료를 제거하는 동작입니다.


제거하는 동작에는 모터가 오랫동안 회전하여 재료를 전체 다 제거합니다. 또한 이 긴 시간동안 모터만 작동시키게 할 수는 없습니다. 온도도 확인해야하고, 디스플레이에 표시도 해야되기 때문입니다.

그렇기 때문에, 현재 상태파악하고 모터를 작업시킬 함수가 하나 더 필요합니다.

setMotorSpeed 는 기존에 선언된 것으로 재료리스트의 속도 값이 저장되는 변수입니다.

timeWork 변수는 시간을 측정하는데 사용합니다. 예를 들어 아래와 같은 코드가 있다면, timeWork이 0이 되기 까지 1초가 걸립니다.
 1int timeWork = 20;

212
213            result = temptable[i][CELSIUS_TEMPTB] - ratioTemp*(temptable[i][CELSIUS_TEMPTB] - temptable[i-1][CELSIUS_TEMPTB]);
214
215            if(result < 10)
216            {

218            }
219            else if(result < 100)

226            }

231        }
232    }

234    return ;

237float Kp=15, Ki=1, Kd=1.0, dT = 0.05;
238int16_t error, previousError = 0; // 오차 변수
239float integral, derivaitve = 0;   // 적분 미분 변수
240
241/*
242* pid 계산을 하고, 결과값을 HEATER_EN 핀에 적용
243*/
244void getPIDoutput(int targetTemp, int actualValue)
245{
246    float outputValue;
247
248    // 에러 값 저장
249    error      = targetTemp - actualValue;
250
251    // 적분 값 저장
252    integral   = integral + (float)error*dT;
253
254    // 미분 값 저장
255    derivaitve = ((float)error - (float)previousError)/dT;
256    previousError = error;
257
258    // PID 계산
259    outputValue = Kp*error + Ki*integral + Kp*derivaitve;
260
261    // output 값이 0~255 범위를 벗어나면 최대, 최소 값을 대신 저장
262    if(outputValue > 255)
263    {
264        outputValue = 255;
265    }
266    else if(outputValue <0)
267    {
268        outputValue = 0;
269    }
270
271    // outputValue 가 0이라면 예열이 되지 않음으로, isHeating 변수를 false로 저장
272    if(outputValue == 0)
273    {
274        isHeating = false;
275    }
276    else
277    {
278        isHeating = true;
279    }
280
281    // 계산된 결과값을 디지털 9번핀에 analogWrite 로 입력
282    analogWrite(HEATER_EN,outputValue);
283}

297        }
298    }
299    // B버튼이 눌리고, A버튼이 눌리지 않았을 경우 실행
300    else if(digitalRead(BTN_A) && !digitalRead(BTN_B))
301    {
302        // B 버튼이 눌러져 있으면(isPressedB_BTN 가 true), 아래 코드 건너뜀
303        if(!isPressedB_BTN)
304        {
305            // 동작 상태를 필라멘트 제거로 변경하고,
306            // 시간을 10초 동안 작동하도록 timeWork 변수에 값 저장
307            stateAction = REVERSE;
308            timeWork = 200;
309        }
310    }
311    else
312    {
313        // A, B 버튼 모두 눌리지 않을 경우, 변수 값 변경
314        isPressedA_BTN = false;
315        isPressedB_BTN = false;
316
317        // 동작 상태를 리렉션으로 변경하고,
318        // 시간을 1초 동안 작동하도록 timeWork 변수에 값 저장
319        if(stateAction == EXTRUDE)
320        {
321            stateAction = RETRACT;
322            timeWork = 20;
323        }
324    }
325
326
327    // C버튼이 눌리고, D버튼이 눌리지 않았을 경우 실행
328    if(!digitalRead(BTN_C) && digitalRead(BTN_D))
329    {
330        // C 버튼이 눌러져 있으면(isPressedC_BTN 가 true), 아래 코드 건너뜀
331        if(!isPressedC_BTN)
332        {
333            // 구조체 배열의 인덱스로 사용될 변수 값 1 증가
334            material_index++;
335            if(material_index > MATERIAL_COUNT-1)
336            {
337                material_index = 0;
338            }
339
340            // 디스플레이 업데이트를 할 수 있도록 변수 변경
341            isNeedUpdateScreen = true;
342
343            // 버튼 상태 변수 변경
344            isPressedC_BTN = true;
345
346            // 적분 값 리셋
347            integral = 0;
348        }
349    }
350    // D버튼이 눌리고, C버튼이 눌리지 않았을 경우 실행
351    else if(digitalRead(BTN_C) && !digitalRead(BTN_D))
352    {
353        // D 버튼이 눌러져 있으면(isPressedD_BTN 가 true), 아래 코드 건너뜀
354        if(!isPressedD_BTN)
355        {
356            // 구조체 배열의 인덱스로 사용될 변수 값 1 감소
357            material_index--;
358            if(material_index < 0)
359            {
360                material_index = MATERIAL_COUNT-1;
361            }
362
363            // 디스플레이 업데이트를 할 수 있도록 변수 변경
364            isNeedUpdateScreen = true;
365
366            // 버튼 상태 변수 변경
367            isPressedD_BTN = true;
368
369            // 적분 값 리셋
370            integral = 0;
371        }
372    }
373    else
374    {
375        // C, D 버튼 모두 눌리지 않을 경우, 변수 값 변경
376        isPressedC_BTN = false;
377        isPressedD_BTN = false;
378    }
379
380}

382/*
383 * 화면에 표시될 재료명, 목표속도, 목표온도를 업데이트
384 */
385void updateMaterial(int index)
386{
387    // isNeedUpdate 변수가 false 이면 함수를 종료함
388    if(!isNeedUpdateScreen) return;
389
390    // 목표 온도, 속도 항목에 현재 인덱스의 구조체 값을 저장
391    setTemperature = materials[index].temperature;
392    setMotorSpeed  = materials[index].motorSpeed;
393
394    // 화면 클리어
395    ssd1306_clearScreen();
396
397    // 화면에 목표 온도, 재료 명, 속도를 표시
398    // 재료명은 좌표 0,0 에 표시
399    // 목표 속도는 좌표 80, 0 에 표시
400    // 목표 온도는 좌표 52, 16 에 표시
401    strToShow = materials[index].materialName;
402    showTextToScreen(0,0, strToShow);
403
404    strToShow = String(setMotorSpeed) + "%";
405    showTextToScreen(80,0, strToShow);
406
407    strToShow = "/" + String(setTemperature);
408    showTextToScreen(52,16,strToShow);
409
410    // 화면을 계속해서 업데이트 하지 않도록 방지하는 bool 변수 변경
411    isNeedUpdateScreen = false;
412}

416    pinMode(BTN_A, INPUT_PULLUP);
417    pinMode(BTN_B, INPUT_PULLUP);
418    pinMode(BTN_C, INPUT_PULLUP);

423    pinMode(MOTOR_SPEED, OUTPUT);
424

426

428
429    ssd1306_128x32_i2c_init(); // 32로 변경
430    ssd1306_fillScreen(0x00);  // 화면 초기화
431    ssd1306_setFixedFont(ssd1306xled_font6x8); // 폰트 설정
432    ssd1306_flipHorizontal(1); // x 화면 대칭 회전
433    ssd1306_flipVertical(1);   // y 화면 대칭 회전
434}

436void loop()
437{
438    if(millis() - timeInterval > 50)
439    {
440        curTemp = getTemperature();
441        updateMaterial(material_index);
442        checkBtnPressed();
443        getPIDoutput(setTemperature, curTemp);
444        checkForReadyToExt();
445        workAction();
446
447        timeInterval = millis();
448    }
449}
