Step.2 모터 작동 시키기


● 시작 블록을 클릭합니다.
● q 키를 눌렀을 때 블록을 드래그하여 이동시켜 줍니다.



● 하드웨어 블록을 클릭합니다.



● 먼저 모터를 On, Off 하는 블록부터 만들어 보겠습니다. 디지털 0번 핀 켜기 블록을 이동시킵니다.



● 디지털 0번 핀 켜기 를 숫자 0을 5로 변경시켜줍니다.
● 디지털 5번 핀은 모터의 On,Off를 담당하는 부분에 연결되어 있기 때문입니다. 이 부분은 제작하는 사람이 결정하는 것이라 숫자에는 크게 신경안쓰셔도 됩니다.
● 디지털 5번 핀이 켜지면 모터가 작동준비를 하게 되고, 끄기를 선택하면, 방향, 속도가 정해져도 작동하지 않습니다.



● 이제 두번째 부분인 회전 방향을 담당하는 블록도 만들어 주기 위해 디지털 0번 핀 켜기 블록을 하나 더 이동시켜줍니다.



● 디지털 0번 핀 켜기 를 숫자 0을 6으로 변경시켜줍니다. 디지털 6번 핀은 모터의 방향을 결정합니다. 키고 끄는 것에 따라 반시계, 시계 방향으로 회전합니다.



● 마지막으로 속도를 결정하는 블록을 만들기 위해 디지털 3 번 핀을 255 (으)로 정하기 블록을 이동시켜줍니다.



● 디지털 3 번 핀을 255 (으)로 정하기 에서 숫자 3을 10으로 변경해줍니다.
● 이어서 뒷편에 있는 255는 속도를 뜻합니다. 0~255까지 사용할 수 있으며, 255는 최대 속도, 0은 최소 속도가 되겠습니다.



● 시작하기 버튼을 누른 뒤에 Q 버튼을 눌러 모터의 움직임을 확인해 봅니다.



● 모터가 움직이나요? 그렇다면 다양한 방법으로 모터를 움직여 봅시다.

1. 모터가 정지하는 블록
2. 모터가 반시계 방향으로 회전하는 블록
3. 모터가 시계 방향으로 회전하는 블록-속도 느리게 해보기
