Step.2 모터 작동 시키기

3D 펜의 버튼을 눌러 모터를 회전 시켜볼 계획입니다. 코딩 맛보기에서 버튼을 사용하기 위해 만들었던 코드 블록들을 다시 준비합니다.


● 시작 블록으로 이동합니다.



● 시작하기 버튼을 클릭했을 때 를 가져옵니다.



● 흐름 블록으로 이동합니다.



● 계속 반복하기 블록을 가져옵니다.



● 만약 참 (이)라면 ~ 블록을 계속 반복하기 블록 안에 이동시켜줍니다.



● 하드웨어 블록을 클릭합니다.



● A 버튼이 눌렸는가? 블록을 만일 참 (이)라면 ~ 에서 참 부분에 이동시켜줍니다.
● 버튼이 눌렸으면 아래 블록을 실행하게 합니다.



● 먼저 모터를 On, Off 하는 블록부터 만들어 보겠습니다. 모터 활성화 블록을 이동시킵니다.
● 모터 활성화 블록은 모터를 On 해주는 블록입니다.



● 모터를 반시계 방향으로 설정 블록을 가져옵니다.
● 이 블록은 모터가 회전하는 방향을 설정해줍니다. 시계, 반시계 중 원하는 방향을 설정합니다.



● 모터 속도를 255으로 설정 블록을 가져옵니다.
● 이 블록은 모터의 회전 속도를 설정합니다 최소 0 에서 최대 255까지 설정할 수 있습니다.



● 시작하기 버튼을 누르고 3D 펜이서 A 버튼이 눌러 봅니다.


● 모터가 움직이나요? 그렇다면 다양한 방법으로 모터를 움직여 봅시다.

1. B 버튼을 누르면, 모터가 정지
2. C 버튼을 누르면, 모터가 반대 방향으로 회전
3. D 버튼을 누르면, 모터 속도가 절반으로 줄어들기
