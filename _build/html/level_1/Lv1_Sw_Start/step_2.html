Step.2 설정

설치가 완료되었으면, 3D 펜과 컴퓨터가 연결되도록 설정을 해주어야 합니다.
아래 과정을 진행해주시기 바랍니다.

● 설치 완료하면, 실행이 됩니다. 여기서 아두이노 나노를 찾고 클릭해줍니다.
● 찾아도 보이지 않는다면, 오른쪽 상단에 검색창에 ‘아두이노’를 검색합니다.



● USB 케이블을 이용하여, 컴퓨터와 3D 펜의 보드를 연결시켜줍니다.



● 왼편의 ‘드라이버 설치’ 버튼을 클릭해줍니다.



● 드라이버 설치 창이 나타나고, 여기에서 ‘다음’ 버튼을 클릭해줍니다.



● 설치가 완료되면, ‘마침’ 버튼을 클릭해줍니다.



● 오른편의 ‘펌웨어 설치’ 버튼을 클릭해줍니다.



● 설치가 완료되면, 위 사진과 같이 ‘연결 성공’이라는 표시가 나타납니다.



● 이제 다시 엔트리로 이동합니다.
● 이동하고, 이 화면에서 ‘하드웨어 연결하기’ 버튼을 눌러줍니다.
● 여기까지 진행하였으면, SW 코딩을 하기 위한 설정은 완료됩니다.





※ 다시 진행하려 할 경우에는 다음과 같은 순서대로 진행해주시면 됩니다.

● 하드웨어 블록을 클릭합니다.



● ‘연결 프로그램 열기’ 버튼을 클릭합니다.



● 아두이노 나노를 찾고 클릭해줍니다.



● 아두이노 나노를 찾고 클릭해줍니다.
● 이후 연결 성공이 되면, 네이버 엔트리 사이트로 돌아옵니다.
