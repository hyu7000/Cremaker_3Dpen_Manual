프로그램 설치

3D 모델링 프로그램은 대부분 조작, 사용방법이 비슷합니다.
기능이 더 좋고 많은 것에는 유료 프로그램도 있지만 초보자 입장에서는 무료프로그램으로도 충분히 여러 프로젝트를 수행할 수 있습니다.

여기서 소개드릴 프로그램은 FreeCad로 이름부터 무료처럼 보입니다.
무료이지만 있을 기능들은 다 있으며, 나름 훌륭한 프로그램입니다.


● 구글에서 FreeCad를 검색합니다.
● 첫번째 사이트 FreeCad에 접속합니다.


● 접속한 사이트에서 download now 를 선택합니다.


● 설치 파일을 실행하고, 나타나는 첫 화면에서 Next 버튼을 눌러줍니다.


● 이어서 Next 버튼을 눌러줍니다.

● 이어서 Next 버튼을 눌러줍니다.


● 이어서 Install 버튼을 눌러줍니다.


● 이후 기다리면, 설치가 완료됩니다.
