메인 케이스 상단 디자인


네 번째로 진행해볼 부품은 3D 펜의 메인 케이스중 하단부품입니다.
디자인은 위와 같습니다. 4개의 부품 중에 가장 복잡해보입니다만 시작해보면 크게(?) 어렵진 않습니다.
시작해보겠습니다.

Step.1
FreeCad 프로그램을 실행해줍니다.


● 새로 만들기 버튼을 눌러줍니다.

Step.2

● 화면이 바뀌면 상단의 Start 버튼을 Part Design 으로 변경해줍니다.

Step.3

● XY 평면에 위와 같은 스케치를 작성해 줍니다.
● x,y,z 축과 만나는 선을 그릴 때는 먼저 선을 다른 곳에 그리고, ‘점을 선에 일치’ 버튼으로 붙여줍니다.
● 작성 후 스케치를 빠져나옵니다.

Step.4

● 회전 버튼을 눌러줍니다.
● 축 항목을 ‘절대좌표계 Y축’ 으로 변경해줍니다.
● 각도 항목을 180°로 입력합니다.
● 작업을 편하게 하기 위해 Reverse 항목을 체크하여, 아래쪽으로 회전되도록 합니다.
● 확인 버튼을 눌러줍니다.

Step.5

● 다시 XY 평면에 위와 같은 스케치를 작성해 줍니다.
● 왼편, 오른편은 서로 대칭입니다.
● 작성 후 스케치를 빠져나옵니다.

Step.6

● 돌출 컷 버튼을 눌러줍니다.
● Length 항목 3.75mm 를 입력합니다.
● 확인 버튼을 눌러줍니다.

Step.7

● XZ 평면에 위와 같은 스케치를 작성해 줍니다.
● 스케치 그리기 전 View section 기능을 사용해줍니다.
● 닫기 버튼을 눌러 스케치를 빠져나옵니다.

Step.8

● 돌출 버튼을 눌러줍니다.
● Length 항목은 78mm 를 입력합니다.
● 방향이 다르다면 Reverse 항목을 체크해줍니다.
● 확인 버튼을 눌러줍니다.

Step.9

● 다시 XZ 평면에 위와 같은 스케치를 작성해 줍니다.
● 스케치 그리기 전 View section 기능을 사용해줍니다.
● 닫기 버튼을 눌러 스케치를 빠져나옵니다.

Step.10

● 돌출 버튼을 눌러줍니다.
● Length 항목은 8mm 를 입력합니다.
● 방향이 다르다면 Reverse 항목을 체크해줍니다.
● 확인 버튼을 눌러줍니다.

Step.11

● 다시 XZ 평면에 위와 같은 스케치를 작성해 줍니다.
● 스케치 그리기 전 View section 기능을 사용해줍니다.
● 닫기 버튼을 눌러 스케치를 빠져나옵니다.

Step.12

● 돌출 컷 버튼을 눌러줍니다.
● Length 항목은 8mm 를 입력합니다.
● 방향이 다르다면 Reverse 항목을 체크해줍니다.
● 확인 버튼을 눌러줍니다.


여기에서는 튜브가 들어가는 공간을 만들어 줘야 합니다. 기울어진 홈이 있기 때문에 선과 평면을 만들어 줘야 합니다.

Step.13

● YZ 평면에 위와 같은 스케치를 작성해 줍니다.
● 스케치 그리기 전 View section 기능을 사용해줍니다.
● 닫기 버튼을 눌러 스케치를 빠져나옵니다.

Step.14

● 평면 생성 버튼을 눌러줍니다.
● 그림과 같이 가운데 원형 모양을 선택해줍니다.

Step.15

● 평면이 뒤로 살짝 이동하도록 왼쪽 창의 항목 중 In Z-direction 을 -5mm 로 입력합니다.
● 확인 버튼을 눌러줍니다.

Step.16

● 생성한 평면에 스케치를 생성해주고, 그림과 같이 스케치를 작성합니다.
● 스케치 그리기 전 View section 기능을 사용해줍니다.
● 닫기 버튼을 눌러 스케치를 빠져나옵니다.

Step.17

● 돌출 컷 버튼을 눌러줍니다.
● Length 항목은 46.5mm 를 입력해줍니다.

Step.18

● Direction/edge 항목은 select reference.. 로 변경해줍니다.
● step.13에서 만든 선을 선택합니다.
● 잘 보이지 않는다면, 보기 - 그리기 스타일 - 와이어 프레임을 선택해줍니다.
● 방향이 달라서 적용이 안된 것 처럼 보일 수 있으니 Reverse 항목을 체크해줍니다.
● 확인 버튼을 눌러줍니다.

Step.19

● 작업 편의를 위해 평면을 숨겨줍니다.
● 생성된 평면을 선택합니다.
● 보기 - 표시 여부 - 선택영역 숨기기를 선택해줍니다.

Step.20

● XZ 평면에 위와 같은 스케치를 작성해 줍니다.

● 방향이 다르다면 Reverse 항목을 체크해줍니다.
● 확인 버튼을 눌러줍니다.
